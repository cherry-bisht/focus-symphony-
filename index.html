How to Conduct

Follow these steps to set up Focus-Symphony on your machine:

  1. Clone the Repo: git clone https://github.com/focus/symphony.git
  2. Build with Go: go build -o focus-symphony main.go
  3. Initialize: Run ./focus-symphony start to begin your focus session.

The tool will automatically block distracting sites by modifying your /etc/hosts file and start your curated focus playlist.

A Modern Composition

Focus-Symphony treats your kernel like a concert hall. By tapping directly into the /proc filesystem, it achieves near-zero latency while maintaining a whisper-quiet footprint.

  • Compiled Golang Core
  • Native Arch Linux Optimization
